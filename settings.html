Google Drive (invoice documents)

Supplier invoices are stored in your Google Drive under Year → Month folders.

Personal Gmail: use OAuth (below). Service accounts cannot upload to My Drive — they have no storage quota.

One-time OAuth setup

  1. Google Cloud Console → APIs & Services → Credentials → create OAuth client ID (Web application).
  2. Add redirect URI: https://developers.google.com/oauthplayground
  3. Open OAuth Playground → gear icon → check Use your own OAuth credentials → enter Client ID & Secret.
  4. Select scope https://www.googleapis.com/auth/drive → Authorize with your Gmail → Exchange authorization code for tokens → copy the Refresh token.
  5. In Supabase → Edge Functions → Secrets, set: GOOGLE_OAUTH_CLIENT_ID, GOOGLE_OAUTH_CLIENT_SECRET, GOOGLE_OAUTH_REFRESH_TOKEN.
  6. Create a folder in Drive, copy its ID from the URL (…/folders/FOLDER_ID), paste below.
