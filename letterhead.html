How to use

Letterhead uses the same station branding as invoices and reports.

Blank stationery

Leave subject and body empty, then use Print blank or Download blank. Useful for handwritten notes or filling later in Word. Blank sheets are not added to history.

Typed letter

Enter an optional date and subject, type the body, then Print letter or Download letter. A signature block is added automatically. Each typed print or Word download is saved under Letter history.

Letter history

Open a past letter to edit and reprint, or print it again as saved. Only admins can delete history rows.

PDF files

There is no separate PDF download. Use Print → Save as PDF in the browser print dialog (same pattern as invoices and reports).

Update branding

Change name, address, GSTIN, or contact details in Settings → Station & branding. The letterhead updates on the next load.
